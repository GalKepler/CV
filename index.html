Selected Projects

NeuroAlign — regional brain-age embeddings & LLM interpretation

Python · scikit-learn · LLM evaluation  |  github.com/GalKepler/NeuroAlign

Extends brain-age prediction to the regional level, embedding each participant as a vector of regional brain-age-gap values for nearest-neighbor retrieval and LLM-generated interpretation. Built with a deterministic evaluation harness to keep LLM output grounded in the underlying data rather than free-associating.

BrainLink — research data infrastructure for SNBB

Python · FastAPI · SQLAlchemy · Docker  |  github.com/GalKepler/brainlink

Unifies multimodal research data — demographics, questionnaires, and MRI file paths — from Google Sheets and BIDS directories into a single queryable SQLite database, with a FastAPI dashboard and scheduled auto-ingest. Runs in production as the data backbone of the ~4,500-participant SNBB dataset.

regional-stacker — bias-corrected regional prediction

Python · scikit-learn

sklearn-compatible package (RegionalStacker, AgeBiasCorrectedRegressor) for nested cross-validation with parallel per-region model fitting and age-bias correction for observational neuroimaging data. Private repo.

KePrep — dMRI preprocessing pipeline

Python · Nipype/niworkflows  |  github.com/GalKepler/keprep

Minimal, reproducible dMRI preprocessing for SNBB — safe copy/retry on network storage, BIDS conversion, and QC, packaged with CI and hosted docs.

fsatlas — automated FreeSurfer atlas morphometry

Python · FreeSurfer  |  github.com/GalKepler/fsatlas

Automates atlas-based cortical/subcortical morphometry extraction across 31 built-in atlases and 4 atlas formats, with parallel batch processing over full BIDS derivative trees.
